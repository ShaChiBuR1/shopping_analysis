# Анализ покупательского поведения

В проекте проведён анализ данных о покупательском поведении клиентов интернет-магазина.

## Содержание

- `data/shopping_behavior.csv` — исходный датасет.
- `notebooks/shopping_analysis.ipynb` — Jupyter Notebook с анализом.
- Анализ включает:
  - Проверку структуры данных и типов
  - Описательную статистику
  - Корреляционный анализ
  - Визуализацию данных (гистограммы, boxplot, heatmap)
  - Группировки и сводные таблицы
- Дашборд в Yandex DataLens (опционально)

## Используемые технологии

- Python 3, Pandas, NumPy, Matplotlib, Seaborn
- Yandex DataLens
